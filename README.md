## Description

A Front-End that connects to a Back-End that communicates with an API, which sends the data back to the Back-End to be interpretted and spit back out to the Front-End, I understand the concept but have not gotten the app to become functional yet. Had lots of problems with errors and even understanding express.
When the app is complete it will allow for the user to find out what the next five days of weather are forecasted for their selected area, Most likely would be used by people who live in cities where the weather is very irratic. I am learning a lot about connecting the back end to the API server and how they all work together.
